# React Table Task\n\nFeatures implemented:\n- Table with dummy data\n- Pagination (10 rows per page by default)\n- Global filtering (search)\n- Column sorting (asc / desc)\n- Responsive layout (desktop + mobile)\n- Download filtered data to Excel (.xlsx)\n- Form to add rows, with validation (React Hook Form + Yup)\n- Global state stored with Redux Toolkit\n\nHow to run:\n1. npm install\n2. npm start\n\nLibraries used:\n- react-table (table, sorting, filtering, pagination)\n- @reduxjs/toolkit + react-redux (global state)\n- react-hook-form + yup (form + validation)\n- xlsx + file-saver (export to Excel)\n
